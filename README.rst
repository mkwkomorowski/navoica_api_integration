navoica_api_integration plugin for `Tutor <https://docs.tutor.overhang.io>`__
===================================================================================

Installation
------------

::

    pip install git+https://plisi-gitlab.opi.org.pl/opi-pib/navoica_api_integration.git

Usage
-----

::

    tutor plugins enable navoica_api_integration
